Acerca de esta aplicación

Descripción

La aplicación "Países del Mundo" es una Single Page Application (SPA) desarrollada con Angular que proporciona información detallada sobre países de todo el mundo. Con esta aplicación, los usuarios pueden explorar datos relevantes sobre diferentes países, como su capital, población, bandera y más.

Características principales

  • Exploración por regiones
  • Búsqueda avanzada
  • Información detallada
  • Interfaz intuitiva y fácil de usar

Desarrollo

Esta aplicación fue desarrollada como parte de un proyecto de clase utilizando las tecnologías modernas de desarrollo web, incluyendo Angular, Bootstrap y la API Rest Countries para obtener datos en tiempo real sobre los países.
