Bienvenido a CountryApp

CountryApp es una plataforma para explorar información detallada sobre países de todo el mundo.

En esta aplicación, puedes:

  • Buscar países por su nombre, capital o región.
  • Explorar detalles de cada país, incluyendo su bandera, población, idioma, moneda y más.
  • Aprender sobre diferentes culturas y geografías del mundo.

CountryApp está desarrollada con Angular y utiliza la API Rest Countries para obtener datos sobre países.
